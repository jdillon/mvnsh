Building
--------

### Requirements

* [Maven](http://maven.apache.org) 3+
* [Java](http://java.sun.com/) 6+

Check-out and build:

    git clone git://github.com/jdillon/mvnsh.git
    cd mvnsh
    mvn install

Extract the assembly and execute the shell:

    unzip mvnsh-assembly/target/mvnsh-*-bin.zip
    ./mvnsh-*/bin/mvnsh
